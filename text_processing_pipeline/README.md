# 🧠 NLP Text Preprocessing Pipeline (Streamlit App)

A complete and interactive Natural Language Processing (NLP) project that performs text preprocessing using a custom pipeline.  
The project is built with **Python, spaCy, and Streamlit** and is designed for real-world NLP tasks such as cleaning, tokenization, stopword removal, lemmatization, and comparison with stemming.

---

## 🚀 Features

This project includes the following functionalities:

### 🔹 Text Preprocessing Pipeline
- Contraction expansion (e.g., "don't → do not")
- Text cleaning (lowercasing, removing special characters, numbers)
- Custom tokenization using regex
- Stopword removal
- Lemmatization using **spaCy**

---

### 🔹 NLP Comparisons
- 🌱 Stemming using PorterStemmer (NLTK)
- 🌿 Lemmatization using spaCy
- Side-by-side comparison of both methods

---

### 🔹 Batch Processing
- Upload CSV dataset
- Process full text column
- Download processed output as CSV

---

### 🔹 Streamlit Web Interface
- User-friendly UI
- Real-time text processing
- Interactive results display
- Clean sentence output format
