# 📊 Model Evaluation Dashboard

## 📝 Objective
This project builds an interactive dashboard to evaluate and compare machine learning models using key performance metrics such as Accuracy, Precision, Recall, and F1-score.

---

## 🚀 Features

### ✅ Core Features
- Train multiple ML models:
  - Logistic Regression
  - SVM
  - Naive Bayes
- Evaluate models using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
- Interactive dashboard using Streamlit

---

### 🔥 Bonus Features
- 📌 Confusion Matrix visualization
- 📈 ROC Curve with AUC
- 📥 Export results as CSV
- 📂 Upload custom dataset
- ⚖ Compare models side-by-side
- 🌙 Dark mode UI
